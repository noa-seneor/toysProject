Toys API documentation

Method: GET
  • /toys get 10 toys per page
  • /toys?page=2 get second page of toys list
  • /toys?perPage=5 get 5 toys per page
  • /toys/search?s=doll search toys by name or category
  • /toys/prices?min=10&max=40 search toys selecting min/max price (s)
  • /toys/category/:catName search toys by category
  • /toys/single/:toyId search toy by it's id

Method: POST
/toys
Only for logged in user, must send auth token through header
  • name (required)
  • info (required)
  • category (required)
  • img_url
  • price (required)

Method: PUT
/toys
Only for logged in user, must send auth token through header
  • name (required)
  • info (required)
  • category (required)
  • img_url
  • price (required)

Method: DELETE
/toys/:toyId: Only for logged in user, must send auth token through header, delete toy by it's id token must correspond to user_id of the toy
